# Chicken Cross

A modern, infinite-scrolling arcade game inspired by Crossy Road and Frogger.

Chicken Cross is a web-based arcade game where you guide a chicken across an endless series of busy roads. The goal is simple: survive as long as possible and achieve the highest score.
